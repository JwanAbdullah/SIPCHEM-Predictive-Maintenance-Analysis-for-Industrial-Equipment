# Exploratory Data Analysis (EDA)

## Objective

The objective of this phase is to explore the cleaned predictive maintenance dataset to identify patterns associated with machine failures. The analysis focuses on comparing failure rates across machine characteristics and operational conditions.

---

## Analysis Performed

### 1. Machine Failures by Machine Type

Purpose:
Determine whether certain machine types experience failures more frequently than others.

Visualization:
- Count Plot
- Failure Percentage Table

Key Insight:
Summarize which machine type has the highest failure rate.

---

### 2. Failure Rate by Tool Wear Category

Tool Wear Categories

- Low
- Medium
- High

Purpose:
Evaluate whether increasing tool wear is associated with higher machine failure rates.

Visualization:
- Bar Chart

Key Insight:
Describe the observed trend.

---

### 3. Failure Rate by Rotational Speed Category

Speed Categories

- Low RPM
- Medium RPM
- High RPM

Purpose:
Assess how rotational speed relates to equipment failure.

Visualization:
- Bar Chart

Key Insight:
Summarize findings.

---

### 4. Failure Rate by Torque Category

Torque Categories

- Low
- Medium
- High

Purpose:
Investigate whether machines operating under higher torque experience more failures.

Visualization:
- Bar Chart

Key Insight:
Summarize findings.

---

### 5. Temperature Comparison

Variables

- Air Temperature
- Process Temperature

Groups

- Failed Machines
- Non-Failed Machines

Purpose:
Compare operating temperatures between failed and non-failed machines.

Visualization

- Boxplots
- Summary Statistics

Key Insight

Discuss whether temperature differences appear significant.

---

## Summary

The exploratory analysis provides an initial understanding of how operational variables influence machine failure. These findings will support the next phase of risk categorization and predictive maintenance recommendations.
