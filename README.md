# Predictive Maintenance Analysis for Industrial Equipment

## Overview
This project is one of the technical tasks completed during my internship at **SIPCHEM**.

The project analyzes the **AI4I 2020 Predictive Maintenance Dataset** to identify machine failure patterns, calculate operational Key Performance Indicators (KPIs), classify equipment risk, and generate business insights for preventive maintenance planning.

The project is implemented in **Python** using **Pandas** and follows a structured data analytics workflow consisting of data preparation, KPI calculation, exploratory data analysis, risk scoring, and visualization.

---

## Objectives

- Prepare and validate the dataset.
- Calculate operational KPIs.
- Analyze machine failure patterns.
- Develop a machine risk scoring model.
- Visualize maintenance insights.
- Provide business recommendations for preventive maintenance.

## Dataset

**Dataset:** AI4I 2020 Predictive Maintenance Dataset

The dataset contains approximately **10,000 machine operating records** with variables including:

- Machine Type
- Air Temperature
- Process Temperature
- Rotational Speed
- Torque
- Tool Wear
- Machine Failure
- Failure Mode Indicators

## Installation

Clone the repository:

```bash
git clone https://github.com/JwanAbdullah/SIPCHEM-Predictive-Maintenance-Analysis-for-Industrial-Equipment.git
```

Create a virtual environment:

```bash
python -m venv .venv
```

Activate the environment.

**Windows**

```bash
.venv\Scripts\activate
```

Install the required packages:

```bash
pip install -r requirements.txt
```

---

## Running the Project

Execute:

```bash
python src/main.py
```
